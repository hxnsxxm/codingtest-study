<h2> ✔️ 참여 방법</h2>


1. 해당 저장소 `fork`
2. 생성된 원격 저장소에 `이름` 혹은 `github ID`로 폴더 생성
3. 생성된 폴더에 자신의 소스코드 업로드
4. `commit 규칙`을 지켜 저장소에 커밋, 푸시

<br>

<h2>✔️ commit 규칙
</h2>

* [문제 출처(플랫폼)] 문제번호 / 문제이름 / 난이도

  ```bash
  git commit -m "[BOJ] 2557 / Hello World / 브론즈5"
  ```

* 플랫폼 작성법

  * [BOJ] - 백준
  * [PGS] - 프로그래머스
  * [ETC] - 그외 (리드미 수정 알고리즘 코드 커밋 이외의 작업 등)
